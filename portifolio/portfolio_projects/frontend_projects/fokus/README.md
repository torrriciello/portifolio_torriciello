# ⏱️ Fokus - Interface de Produtividade

O **Fokus** é uma interface web que ajuda nos estudos e na organização do tempo utilizando a técnica Pomodoro. O usuário pode alternar entre foco, descanso curto e descanso longo, além de gerenciar tarefas durante as sessões.

---

## 🚀 Funcionalidades

- ⏳ Temporizador estilo Pomodoro (foco e pausas)
- 🎯 Modos de foco, descanso curto e descanso longo
- 🎵 Música ambiente opcional
- 📝 Lista de tarefas (CRUD completo)
- ✔️ Seleção de tarefa em andamento
- 💾 Persistência de dados com `localStorage`
- 🧠 Marcação automática de tarefa como concluída ao final do foco
- 🔔 Eventos personalizados com `CustomEvent`

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage
- Custom Events (DOM)

---

## 📂 Estrutura do projeto

- `index.html` → estrutura principal da aplicação
- `styles.css` → estilização da interface
- `script.js` → controle do timer e modos (foco/pausa)
- `script-crud.js` → gerenciamento de tarefas
